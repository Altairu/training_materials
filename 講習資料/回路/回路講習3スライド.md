# 注意点

- 素子は部室にあるもの（主にモータードライバーで使用したもの）
- 回路の大きさは小さければ小さいほど良い
- 電流は最低40A以上耐久（線は太くってこと）
- 24Vから3.3Vの降圧を３端子レギュレーターで行うと熱くなるのでやめましょう．
- マイコンはESP32 C3を使用せよ
（回路講習０で作ったシンボル，フットプリントを用いる．）
